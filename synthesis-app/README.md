# ai2human — Synthesis Hackathon Build

This is a standalone hackathon-focused app for the Synthesis submission.

Positioning:
- ai2human: Human Fallback Infrastructure for Agents
- Primary track: Agents that cooperate
- Secondary tracks: Agents that trust, Agents that pay

Routes:
- / — landing page
- /demo — hackathon demo loop
- /operators — fallback operator pool
- /tasks/[id] — single task detail

Run:
- `cd synthesis-app`
- `npm run dev -- -p 3010`
- `npm run build`
